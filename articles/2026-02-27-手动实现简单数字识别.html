DQN

之前使用Qlearning玩像素鸟，发现怎么样都卡在了10多分，试了各种超参数优化的算法，后面我发现我用的环境给的参数有问题，所以一直训练不好。改用DQN吧

DQN能解决维度灾难的问题，用神经网络来代替原先的Q表，神经网络能拟合输入和输出，假如把游戏图像输入让它预测n个值，这n个值就代表了n个action的得分。但是训练的时候可能不如Qlearning这么直观。

 

神经网络

通常会使用反向传播的算法进行训练，也就是计算损失函数的梯度，沿着梯度反方向对数值进行更新，逐层求导的过程就是反向传播的过程。通常会分为人工神经网络(ANN)和卷积神经网络(CNN)

计算过程

假设图片的28*28=784

输入层：为784*1的矩阵X

隐藏层：设置20个神经元，权重矩阵为20*784的矩阵W，偏置为20*1的矩阵B

输出层：权重矩阵为10*20的矩阵W1，偏置为10*1的矩阵B1

最终会得到一个10*1的矩阵o，这就代表的预测的结果

Sigmoid 函数:σ

学习率η

 

前向传播

Z=WX+B
H=σ(Z)

从输入层到隐藏层，然后就得到了一个20*1的矩阵H，激活函数的作用是让拟合结果具备非线性拟合的能力。

 

Z1=W1*H+B1
o=σ(Z1)

最终得到了一个10*1的矩阵o,这个就是预测值了

 

反向传播

输出层到隐藏层

使用均方误差作为损失函数，l为真实值

其中o_i为o的第i个元素，接下来需要让E变小，开始求梯度，先将公式展开

然后我们需要调整W1使得损失函数最小，也就是求∂E/∂W1

这样子就得到了输出层到隐藏层的权重梯度上面的公式是不是矩阵运算，但矩阵乘玩后的每一个元素都是一个函数，上面的函数就相当于对成完后得到的元素为函数的矩阵进行的求梯度，最终能得到一个10*1的矩阵C1

 

隐藏层到输入层

我们就需要计算∂E/∂W

沿着上一层，最终得到包含H的式子，将H展开H=σ(WX+B)

这个W就是输入层的权重矩阵了，我们需要继续给这个进行

这样子最终能得到一个20*1的矩阵C

 

用同样的方式还可以求出关于B和B1的梯度

∂E/B1=∂E/∂o*∂o/∂Z1*∂Z1/∂B1,其中∂Z1/∂B1为1，所以实际上我们已经算出来了∂E/B1，同理∂E/B也是，所得的矩阵记作D D1

 

 

梯度下降

输出层权重更新

W1=W1-ηC1

B1=B1-ηD1

 

隐藏层权重更新

W=W-ηC

B=B-ηD

 

这样子就实现了一次训练，这个也就是神经网络的原理了，上面这个又称为多层感知机(MLP)

import numpy as np
import copy


def load_mnist_data():
    """加载MNIST数据集并返回训练图像和标签"""
    with np.load("mnist.npz") as f:
        images, labels = f["x_train"], f["y_train"]

    # 转换为Python列表格式（便于纯Python操作）
    images = images.astype("float32") / 255  # 归一化
    images = images.reshape(
        (images.shape[0], images.shape[1] * images.shape[2])
    )  # 展平
    labels = labels.tolist()  # 转换为Python列表

    # 将图像也转换为Python列表的列表
    images_list = images.tolist()

    return images_list, labels


def matrix_multiply(A, B):
    """矩阵乘法：A (m×n) × B (n×p) = C (m×p)"""
    m = len(A)
    n = len(A[0])
    p = len(B[0])

    # 初始化结果矩阵
    C = [[0.0 for _ in range(p)] for _ in range(m)]

    # 矩阵乘法
    for i in range(m):
        for j in range(p):
            for k in range(n):
                C[i][j] += A[i][k] * B[k][j]

    return C


def matrix_add(A, B):
    """矩阵加法：A + B（要求维度相同）"""
    m = len(A)
    n = len(A[0])

    # 初始化结果矩阵
    C = [[0.0 for _ in range(n)] for _ in range(m)]

    # 逐元素相加
    for i in range(m):
        for j in range(n):
            C[i][j] = A[i][j] + B[i][j]

    return C  # 添加了返回值



def matrix_transpose(A):
    """矩阵转置：A (m×n) -> A^T (n×m)"""
    m = len(A)
    n = len(A[0])

    # 初始化转置矩阵
    AT = [[0.0 for _ in range(m)] for _ in range(n)]

    # 转置
    for i in range(m):
        for j in range(n):
            AT[j][i] = A[i][j]

    return AT


def scalar_matrix_multiply(scalar, A):
    """标量与矩阵相乘：scalar × A"""
    m = len(A)
    n = len(A[0])

    # 初始化结果矩阵
    C = [[0.0 for _ in range(n)] for _ in range(m)]

    # 标量乘法
    for i in range(m):
        for j in range(n):
            C[i][j] = scalar * A[i][j]

    return C


def matrix_subtract(A, B):
    """矩阵减法：A - B（要求维度相同）"""
    m = len(A)
    n = len(A[0])

    # 初始化结果矩阵
    C = [[0.0 for _ in range(n)] for _ in range(m)]

    # 逐元素相减
    for i in range(m):
        for j in range(n):
            C[i][j] = A[i][j] - B[i][j]

    return C


def sigmoid(x):
    """Sigmoid激活函数"""
    return 1.0 / (1.0 + 2.718281828459045 ** (-x))


def matrix_sigmoid(A):
    B = copy.deepcopy(A)
    for i in range(len(A)):
        for j in range(len(A[0])):
            B[i][j] = sigmoid(A[i][j])
    return B


images, labels = load_mnist_data()

W1 = [[np.random.uniform(-0.5, 0.5) for _ in range(784)] for _ in range(20)]
b1 = [[0.0] for _ in range(20)]

W2 = [[np.random.uniform(-0.5, 0.5) for _ in range(20)] for _ in range(10)]
b2 = [[0.0] for _ in range(10)]


def train(X, label):
    global W1, W2, b1, b2
    # 输入层 -> 隐藏层
    Z1 = matrix_multiply(W1, X)  # 20×1
    Z1 = matrix_add(Z1, b1)  # 加上偏置
    H1 = matrix_sigmoid(Z1)  # 20×1

    # 隐藏层 -> 输出层
    Z2 = matrix_multiply(W2, H1)  # 10×1
    Z2 = matrix_add(Z2, b2)  # 加上偏置
    O = matrix_sigmoid(Z2)  # 10×1

    # 反向传播
    # 输出层
    L_sub_O = matrix_subtract(O, label)  # 10×1 (L-O)

    O_mul_1_sub_O = copy.deepcopy(O)
    for i in range(len(O)):
        O_mul_1_sub_O[i][0] = O[i][0] * (1 - O[i][0])  # O(1-O) 10x1
    L_sub_O = scalar_matrix_multiply(0.2, L_sub_O)  # 0.2(L-O) 10x1

    dE_dW2 = copy.deepcopy(O)
    for i in range(len(L_sub_O)):
        for j in range(len(L_sub_O[0])):
            dE_dW2[i][j] = L_sub_O[i][j] * O_mul_1_sub_O[i][j]  # 0.2(L-O)O(1-O)    10x1

    dE_dB2 = copy.deepcopy(dE_dW2)

    H1_T = matrix_transpose(H1)  # 1×20
    dE_dW2 = matrix_multiply(dE_dW2, H1_T)  # 10×20
    # print(dE_dW2)

    # 隐藏层
    dE_dO = scalar_matrix_multiply(0.2, matrix_subtract(O, label))  # 0.2(L-o)  10x1

    dO_dZ2 = copy.deepcopy(O)
    for i in range(len(O)):
        dO_dZ2[i][0] = O[i][0] * (1 - O[i][0])  # O(1-O) 10x1

    dZ2_dH1 = matrix_transpose(W2)  # W2    20x10

    dH1_dZ1 = copy.deepcopy(H1)
    for i in range(len(H1)):
        dH1_dZ1[i][0] = H1[i][0] * (1 - H1[i][0])  # H1(1-H1) 20x1

    dZ1_dW1 = matrix_transpose(X)  # 1×784

    dE_dW1 = copy.deepcopy(O)
    for i in range(len(dE_dO)):
        for j in range(len(dE_dO[0])):
            dE_dW1[i][j] = dE_dO[i][j] * dO_dZ2[i][j]  # 0.2(L-O)O(1-O)    10x1

    dE_dW1 = matrix_multiply(dZ2_dH1, dE_dW1)  # 0.2(L-O)O(1-O)*W2 20x1

    for i in range(len(dE_dW1)):
        for j in range(len(dE_dW1[0])):
            dE_dW1[i][j] = (
                dE_dW1[i][j] * dH1_dZ1[i][j]
            )  # 0.2(L-O)O(1-O)*W2*H1(1-H1) 20×1

    dE_dB1 = copy.deepcopy(dE_dW1)
    dE_dW1 = matrix_multiply(dE_dW1, dZ1_dW1)  # 20×784

    # print(dE_dW1)
    # 梯度更新
    # learning_rate = 0.1
    # W1 = matrix_subtract(W1, scalar_matrix_multiply(learning_rate, dE_dW1))
    # b1 = matrix_subtract(b1, scalar_matrix_multiply(learning_rate, dE_dB1))

    # W2 = matrix_subtract(W2, scalar_matrix_multiply(learning_rate, dE_dW2))
    # b2 = matrix_subtract(b2, scalar_matrix_multiply(learning_rate, dE_dB2))
    learning_rate = 0.1
    for i in range(len(W1)):
        for j in range(len(W1[0])):
            W1[i][j] -= learning_rate * dE_dW1[i][j]
    
    for i in range(len(b1)):
        b1[i][0] -= learning_rate * dE_dB1[i][0]
    
    for i in range(len(W2)):
        for j in range(len(W2[0])):
            W2[i][j] -= learning_rate * dE_dW2[i][j]
    
    for i in range(len(b2)):
        b2[i][0] -= learning_rate * dE_dB2[i][0]


def predict(X, label):
    global W1, W2, b1, b2
    # 输入层 -> 隐藏层
    Z1 = matrix_multiply(W1, X)  # 20×1
    Z1 = matrix_add(Z1, b1)  # 加上偏置
    H1 = matrix_sigmoid(Z1)  # 20×1

    # 隐藏层 -> 输出层
    Z2 = matrix_multiply(W2, H1)  # 10×1
    Z2 = matrix_add(Z2, b2)  # 加上偏置
    O = matrix_sigmoid(Z2)  # 10×1
    MAX=-1
    index=0
    for i in range(10):
        if O[i][0]>=MAX:
            index=i
            MAX=O[i][0]

    return index

        label = [[0.0] for _ in range(10)]
        label[labels[j]][0] = 1.0

        # 前向传播
        X = [[i] for i in images[j]]  # 784×1

        # 前向传播
