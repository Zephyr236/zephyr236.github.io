## About Me

This is my personal blog. I write about various topics including technology, mathematics, and more.

## About This Blog

This blog is built with:
- **Jekyll** - Static site generator
- **Markdown** - For writing content
- **LaTeX** - For math formulas (powered by KaTeX)

## How to Write Posts

1. Create a new file in `_posts/` directory
2. Name it with format: `YYYY-MM-DD-title.md`
3. Add front matter at the top:

```yaml
---
layout: post
title: Your Title
date: 2024-01-01
description: Optional description
---
```

4. Write your content in Markdown
5. Use `$...$` for inline math and `$$...$$` for display math

Enjoy!
